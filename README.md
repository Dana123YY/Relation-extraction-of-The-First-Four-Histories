# Dataset of Character Relationships in China's "The First Four Histories"

This dataset is used for the experimental part of the paper *"Research on Graph-Retrieval Augmented Generation Based on Historical Text Knowledge Graphs"*.


## Contents of the Dataset
The project includes the following parts:
1. `train_data` and `test_data` containing reasoning chains for extracting character relationships from historical records
2. `score_data`, a dataset for step-by-step scoring of reasoning chains
3. `KG_data`, a knowledge graph automatically extracted from China's "Four Books of History"
