# Dataset for "Research on Graph-Retrieval Augmented Generation Based on Historical Text Knowledge Graphs"

This dataset is used for the experiments in the paper *"Research on Graph-Retrieval Augmented Generation Based on Historical Text Knowledge Graphs"*.


## Dataset Contents
The project includes the following parts:
1. `train_data` and `test_data`: Containing reasoning chains for entity relation extraction of historical figures from historical records.
2. `score_data`: A dataset for step-by-step scoring of the reasoning chains.
3. `KG_data`: A knowledge graph automatically extracted from the "Twenty-Four Histories" (a collection of Chinese historical records).
